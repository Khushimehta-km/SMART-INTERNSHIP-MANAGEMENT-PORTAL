# Smart Internship Management Portal

A full stack web application developed for managing internship activities.

## About Project

Smart Internship Management Portal helps manage interns, daily work reports, attendance records and feedback in an organized way.

## Features

- Intern Registration and Login
- Role Based Dashboard
- Daily Work Report Submission
- Attendance Management
- Feedback Management
- Dashboard Statistics

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Deployment
- Frontend: Netlify
- Backend: Render
