# Smart Internship Management Portal

## Mindlabz Hospitality Services Private Limited

## Project Description

This project is developed as part of my internship at Mindlabz Hospitality Services Private Limited.

The Smart Internship Management Portal is a web-based application designed to manage internship activities in an organized way.

It allows interns, mentors, and administrators to manage daily work reports, attendance records, feedback, and internship progress through a centralized platform.


## Features Implemented

- User Login and Role Selection (Admin, Mentor, Intern)
- Intern Management
- Daily Work Report Submission
- Attendance Tracking
- Mentor Feedback System
- Progress Dashboard
- Performance Report Generation


## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js

- MongoDB Atlas

- Visual Studio Code
- NPM


## How to Run the Project

1. Open the project folder.

2. Open terminal inside backend folder.

3. Install required packages:

npm install

4. Start backend server:

npm run dev

5. Open frontend/index.html file in browser.

If dependencies are not installed:

npm install

Start server:

npm run dev


## Project Modules

1. Authentication Module

Manages user login and role selection.


2. Dashboard Module

Displays internship progress and statistics.


3. Report Module

Allows interns to submit daily work updates.


4. Attendance Module

Tracks intern attendance records.


5. Feedback Module

Allows mentors to provide comments and ratings.


6. Report Generation Module

Generates internship performance reports.


## Database

MongoDB Atlas is used to store:

- User details
- Work reports
- Attendance data
- Feedback records
